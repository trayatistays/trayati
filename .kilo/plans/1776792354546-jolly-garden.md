# Video Loading Optimization Plan - Mux Streaming

## Overview

Implement Mux video streaming for adaptive bitrate delivery, eliminating lag by streaming only the portions of video needed.

## Why Mux?

- **Adaptive bitrate streaming (HLS)** - Automatically adjusts quality based on user's bandwidth
- **Small initial load** - Starts playing with ~200KB vs downloading full MP4
- **Global CDN** - Fast delivery worldwide
- **Automatic optimization** - Encodes multiple quality levels automatically
- **Analytics** - View video performance metrics

## Implementation Steps

### Step 1: Set Up Mux Account

1. Sign up at [mux.com](https://mux.com)
2. Create a new Environment in the dashboard
3. Get your **Access Token ID** and **Secret Key** from Settings > API Access Tokens

### Step 2: Add Environment Variables

Add to `.env.local`:
```
MUX_ACCESS_TOKEN=your_access_token_id
MUX_SECRET_KEY=your_secret_key
MUX_SIGNING_KEY=your_signing_key (optional, for signed URLs)
MUX_SIGNING_KEY_PRIVATE_KEY=your_private_key (optional)
```

### Step 3: Install Mux SDK

```bash
npm install @mux/mux-player-react
```

### Step 4: Upload Video to Mux

Option A: Upload via Mux Dashboard (Simplest)
1. Go to Mux Dashboard > Video > Assets
2. Click "Create Asset"
3. Upload your hero video or paste the Supabase URL
4. Wait for processing (~1-2 minutes)
5. Copy the **Playback ID**

Option B: Programmatic Upload (for automation)
- Create a video upload API route

### Step 5: Create Video Component

Create `src/components/mux-video.tsx`:
```tsx
"use client";

import MuxPlayer from "@mux/mux-player-react";

interface MuxVideoProps {
  playbackId: string;
  poster?: string;
  className?: string;
}

export function MuxVideo({ playbackId, poster, className }: MuxVideoProps) {
  return (
    <MuxPlayer
      playbackId={playbackId}
      poster={poster}
      muted
      autoPlay
      loop
      playsInline
      streamType="on-demand"
      className={className}
    />
  );
}
```

### Step 7: Optimize Loading (Lazy Load)

Keep the existing desktop-only + delayed loading logic:
- Only load Mux player on desktop (768px+)
- Use Intersection Observer for true lazy loading
- Show poster image while loading

## Implementation Code

### Updated hero-section.tsx

```tsx
"use client";

import Image from "next/image";
import Link from "next/link";
import { useRef, useEffect, useState } from "react";
import MuxPlayer from "@mux/mux-player-react";
import { AnimatedText } from "@/components/animated-text";

const keywords = ["Folklore Homestays", "Villas and Estates", "Apartments"];
const MUX_PLAYBACK_ID = "YOUR_PLAYBACK_ID"; // Replace with actual ID

export function HeroSection() {
  const [shouldLoadVideo, setShouldLoadVideo] = useState(false);
  const [isDesktop, setIsDesktop] = useState(false);
  const sectionRef = useRef<HTMLElement>(null);

  useEffect(() => {
    const mql = window.matchMedia("(min-width: 768px)");
    const handleChange = (e: MediaQueryListEvent | MediaQueryList) => {
      setIsDesktop(e.matches);
    };
    handleChange(mql);
    mql.addEventListener("change", handleChange);
    return () => mql.removeEventListener("change", handleChange);
  }, []);

  // Lazy load video with Intersection Observer
  useEffect(() => {
    if (!isDesktop) return;

    const observer = new IntersectionObserver(
      (entries) => {
        if (entries[0].isIntersecting) {
          setShouldLoadVideo(true);
          observer.disconnect();
        }
      },
      { threshold: 0.1 }
    );

    if (sectionRef.current) {
      observer.observe(sectionRef.current);
    }

    return () => observer.disconnect();
  }, [isDesktop]);

  return (
    <section ref={sectionRef} id="top" className="relative isolate min-h-[85vh]">
      <div className="absolute inset-0 -z-20">
        {/* Poster image fallback */}
        <Image
          src="https://image.mux.com/{MUX_PLAYBACK_ID}/thumbnail.jpg?time=0"
          alt=""
          aria-hidden="true"
          fill
          priority
          fetchPriority="high"
          sizes="(max-width: 768px) 100vw, 85vw"
          className="absolute inset-0 object-cover object-center"
        />
        
        {/* Mux video player */}
        {isDesktop && shouldLoadVideo && (
          <MuxPlayer
            playbackId={MUX_PLAYBACK_ID}
            poster={`https://image.mux.com/${MUX_PLAYBACK_ID}/thumbnail.jpg?time=0`}
            muted
            autoPlay
            loop
            playsInline
            streamType="on-demand"
            className="absolute inset-0 h-full w-full object-cover"
          />
        )}
        
        <div className="absolute inset-0 bg-black/20" />
        <div className="absolute inset-0 bg-[linear-gradient(180deg,rgba(0,0,0,0.1)_0%,rgba(0,0,0,0.35)_60%,rgba(0,0,0,0.55)_100%)] md:hidden" />
      </div>
      
      {/* Rest of hero content... */}
    </section>
  );
}
```
